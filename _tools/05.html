--- title: Adobe Photoshop tool_title: Adobe Photoshop abstract: Adobe Photoshop is raster graphics editing application that supports embedded metadata in image files. Depending on the file format, the metadata is embedded within the file or saved in a sidecar file. Photoshop utilizes Adobe's proprietary metadata standard Extensible Metadata Platform (XMP) which is built on XML. XMP supports Exif, IPTC (IIM), GPS, TIFF, and other embedded metadata. Users can edit individual image files' embedded metadata via a form, a template, or XMP import. There is also an XMP export option.
