--- title: VRA Bridge Metadata Tools abstract: "Javascript extension for Adobe Bridge that provides a cataloging screen and the ability to import and export VRA metadata to a CSV file. It gets installed in the Bridge Startup Scripts. In the Metadata Palette, View and edit embedded metadata for image(s). It's a cataloging screen. In the Metadata Export-Import Tool, export embedded VRA metadata from images to a spreadsheet or import spreadsheet VRA metadata to images. File formats supported: DNG, GIF, JPEG, PNG, PSD, TIFF, RAW." tool_url: http://metadatadeluxe.pbworks.com/w/page/108523528/VRA%20Bridge%20Metadata%20Tools source_code: http://metadatadeluxe.pbworks.com/w/page/108523528/VRA%20Bridge%20Metadata%20Tools creator: Embedded Metadata Working Group, Visual Resources Association type: Plug-in category_broad: - Creation - Editing - Extraction/Embedding category_broad_informal: - Embedded metadata - Create metadata - Import/export metadata use_case: Embed metadata, create metadata, import/export metadata interface: GUI dependencies: Adobe Bridge CS4 and higher (including CC) open_source_proprietary: Open Source cost: Free; MIT License related_tools: VRA Core Transformation Tools (http://core.vraweb.org/vracore_tools.html) metadata_standard: VRA Core, IPTC file_format: CSV ---
