--- title: MatchMARC abstract: MatchMARC is a Google Sheets Add-on created for the Lehigh University Libraries technical services team. It is a tool that automates MARC record searches. It uses OCLC's WorldCat Search API. You can configure the search to return your 'best' match directly within your Google Sheet.
